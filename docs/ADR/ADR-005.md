# ADR-005 — No se modifica código estable únicamente para eliminar advertencias cosméticas

**Estado:** Aceptado

## Contexto

Se detectó una advertencia de consola (`::-webkit-scrollbar` ignorado en navegadores
no-WebKit, en `static/css/styles.css`) sin ningún impacto funcional ni visual — el
comportamiento de scroll seguía siendo correcto en todos los navegadores, la advertencia
era puramente informativa.

## Decisión

No se modifica código estable únicamente para silenciar una advertencia cosmética.

## Por qué

Toda modificación tiene costo: posibilidad de introducir un bug, necesidad de volver a
probar, un commit más, más historial. Si el ROI de arreglar algo es prácticamente cero, no
se toca.

## Excepción implícita

Esta regla aplica a advertencias puramente cosméticas. Una advertencia que sea señal
temprana de algo que se va a romper a futuro (una deprecación, un comportamiento que un
navegador va a dejar de soportar) no cae bajo ADR-005 — eso no es cosmético, es una alerta
real que sí justifica investigar.
