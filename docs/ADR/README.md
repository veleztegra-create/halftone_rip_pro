## Nota sobre ADR-007, 008 y 009

Este índice se generó a partir del historial de decisiones discutidas explícitamente hasta
ahora. En ese historial no hay contenido para ADR-007, 008 ni 009 — ni contexto, ni decisión,
ni justificación. No se inventó contenido para completar el hueco.

Si esas tres decisiones ya existen (en otra conversación, en la cabeza de alguien del equipo,
en un documento no compartido todavía), hay que traer su contenido acá antes de que este
índice se considere completo. Si no existen todavía, la numeración simplemente saltó de
ADR-006 a ADR-010 en algún momento y no hay nada que rellenar — vale la pena decirlo
explícitamente para que no se asuma que hay decisiones "perdidas".
